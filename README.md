# Password Security Toolkit

A Python-based cybersecurity toolkit that helps users generate secure passwords, evaluate password strength, create SHA-256 hashes, and generate a basic security report.

## Features

- Generate secure random passwords
- Check password strength
- Generate SHA-256 hashes
- Generate a security report

## Technologies Used

- Python 3
- secrets
- hashlib
- string

## Installation

```bash
git clone https://github.com/scripted11/Password-Security-Toolkit.git
cd Password-Security-Toolkit
python main.py
```

## Usage

Run the program and choose one of the available options from the menu.
